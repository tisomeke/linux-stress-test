# 🚀 Universal Linux Stress-Tester (Bash)

Универсальный скрипт на Bash для проведения стресс-тестирования процессора (CPU) при помощи загрузки процессом **p7zip** и видеокарты (GPU) при помощи бенчмарка **furmark**. Скрипт поддерживает автоматическое определение оборудования и логирование температур.

## 🛠 Возможности
* **Три режима работы:** Раздельное тестирование CPU или GPU, а также комбинированный тест (`all`) для проверки блока питания.
* **Универсальность:** Автоматический поиск температурных датчиков в `/sys/class/hwmon/` (совместимо с Intel, AMD, NVIDIA).
* **Логирование:** Запись температуры, частоты GPU и системной нагрузки (Load Average) в файл `.log` каждые 10 секунд.
* **Безопасность:** Автоматическое прерывание теста при достижении критических температур (95°C).

## 📦 Установка зависимостей

Для работы скрипта требуются `p7zip`, `furmark` и `nvidia-utils` (для карт NVIDIA).

### Debian / Ubuntu / Mint
```bash
sudo apt update
sudo apt install p7zip-full nvidia-utils-535 lm-sensors
# FurMark устанавливается отдельно (рекомендуется bin-версия или через Flathub)
```

### Fedora / Red Hat / CentOS
```bash
sudo dnf install p7zip p7zip-plugins lm_sensors
# Для NVIDIA (если подключен репозиторий RPM Fusion):
sudo dnf install xorg-x11-drv-nvidia-utils
```

### Arch Linux / CachyOS
```bash
sudo pacman -S p7zip furmark-bin nvidia-utils lm_sensors
```

## 🚀 Использование

0. **Быстрый старт**
    ```bash
    git clone https://github.com/tisomeke/linux-stress-test
    cd linux-stress-test
    ```

1.  **Дайте права на исполнение:**
    ```bash
    chmod +x stress.sh
    ```
2.  **Запустите нужный режим:**

| Команда | Описание |
| :--- | :--- |
| `./stress.sh cpu` | Тест только процессора (5 минут, 100% нагрузка). |
| `./stress.sh gpu` | Тест только видеокарты (бенчмарк FurMark 5 минут). |
| `./stress.sh all` | **Полный стресс-тест 5 минут.** Проверка стабильности БП и системы охлаждения. |

## 📊 Интерпретация логов
Файл лога (`stress_XXXXXX.log`) содержит колонки:
* **Time:** Время с начала теста.
* **CPU Temp:** Температура процессора (норма до 80°C).
* **GPU Temp:** Температура видеокарты (норма до 85°C).
* **Load:** Нагрузка на систему. Значение должно быть близко к количеству потоков вашего CPU.

***

Для изменения времени теста нужно редактировать значение DURATION в скрипте на желаемое значение секунд (по умолчанию установлено 300с).
    ```bash
    DURATION=300 # 5 минут
    ```
